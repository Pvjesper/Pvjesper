### Audio Signal Processing Simulator (ASPSIM)
> **Sound simulation software built on the image-source method of pyroomacoustics**

Built on top of pyroomacoustics for convenient simulation of adaptive sound reproduction.

### Audio Signal Processing Core (ASPCORE)
> **Classic signal processing toolkit**

Includes time-varying MIMO filters for streaming signals, discrete Fourier transforms, adaptive filters, functions for matrices and linear systems, covariance estimation, and more.

[![Code](https://img.shields.io/badge/GitHub-Code-181717?style=flat&logo=github)](https://github.com/sounds-research/aspcore)
